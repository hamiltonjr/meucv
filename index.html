Tecnologias

HTML

Linguagem de Marcação

A linguagem de marcação HTML, em sua versão 5, é essencial para construir páginas web, mesmo que sejam dinamicamente geradas. É no código HTML que o formato da página é determinado.

CSS

Linguagem de Estilo

A linguagem de estilo CSS, na sua versão 3, é fundamental para dar à página web seu estilo (formas, cores, alinhamento). As folhas de estilo devem ser bem desenhadas para que a página fique apresentável.

Javascript

Linguagem de Programação

A linguagem Javascript dá dinamismo à página web. Também permite certa validação de dados inseridos via formulário.

Spring

Framework Java

O Framework Java Spring permite que a páginma web trabalhe como um sistema web, acessando por exemplo um banco de dados e disponibilize informações ao usuário do sistema.

React

Biblioteca Javascript

React é uma bibnlioteca Javascript que atua no front-end da aplicação web de forma componentizada, facilitando o desenvolvimento de interfaces de usuário e tempo recorde.
